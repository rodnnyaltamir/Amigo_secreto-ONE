# Amigo Secreto ✨

## Descripción

Mi nombre es **Altamirano Flores Rodnny Giovani**. Este proyecto es una página web que permite ingresar nombres y luego realiza un sorteo. Es parte del programa de educación de **ORACLE ONE**.

## Características

- Ingreso de nombres de participantes.
- Sorteo aleatorio de "amigo secreto".
- Interfaz amigable y fácil de usar.

## Tecnologías Utilizadas

- HTML
- CSS
- JavaScript

## Cómo Ejecutar el Proyecto

1. Clona el repositorio.
2. Abre el archivo `index.html` en tu navegador web.
